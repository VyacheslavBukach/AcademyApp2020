# AcademyApp2020

## About

A working prototype of the Android application that can show a list of popular movies and it's details from The Movie Database API (TMDb).

## Features

- DI Hilt
- MVVM architecture
- Landscape and portrait orientation
- Kotlin Coroutines
- Paging 3 Library
- Works without Internet (Cache)
